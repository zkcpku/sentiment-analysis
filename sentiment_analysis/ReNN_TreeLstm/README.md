PyTorch implementation of <br>
1) RNN: Recursive Neural Network from https://nlp.stanford.edu/~socherr/EMNLP2013_RNTN.pdf<br>
   python RecursiveNN.py<br>
2) TreeLSTM from https://arxiv.org/abs/1503.00075 <br>
   python TreeLSTM.py<br>

add cuda as an argument to run it using cuda<br>

Requirements:<br>
nltk <br>
pytorch <br>
progressbar <br>

Socher's cs224d class (see for e.g. https://github.com/kingtaurus/cs224d/tree/master/assignment3)<br>
